# FlowBoard

A lightweight Kanban board application inspired by Trello. Built with React and TypeScript. No backend requiredâ€”tasks are persisted in localStorage.

## Features
- Three fixed columns: To Do, In Progress, Done
- Add, move (drag-and-drop or buttons), and delete tasks
- All state managed in React components
- No external drag-and-drop libraries

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
